# [Taller DAWE Autoidentificacion 2017]()

Proyecto correspondiente al curso de Desarrollo de Aplicaciones Web de la Universidad Catolica del Norte 

## Descargar :

```bash
$ npm install
```

## Compilar Angular 4:

```bash
$ ng build
```
## Correr Servidor:

```bash
$ nnpm start
```
